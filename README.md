# 🃏 Blackjack Game

A beginner-friendly Blackjack game built using **HTML**, **CSS**, and **JavaScript**. This project simulates the basic rules of Blackjack, allowing players to start a game, draw new cards, and determine whether they have won, lost, or should continue playing.

---

# ✨ Features

*  Start a new Blackjack game
*  Random card generation
*  Automatic sum calculation
*  Win when the total reaches 21
*  Lose when the total exceeds 21
* Draw additional cards
*  Simple and responsive interface

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
